# Installation Commands for Client Side (React) and Server Side (Express.js)

## Client Side - React

Install the following packages for the React frontend:

```bash
# Create React App
npx create-react-app my-app
cd my-app

# React Router DOM
npm install react-router-dom

# React Bootstrap
npm install react-bootstrap bootstrap

# Axios for API Requests
npm install axios

# React Toastify for Notifications
npm install react-toastify

# SweetAlert 2
npm install react-bootstrap-sweetalert

# Cogo Toast for Alerts
npm install cogo-toast

# Recharts for Charts
npm install recharts

# LocalForage for Local Storage
npm install localforage

# React Export Excel
npm install react-export-excel

# Cleave.js for Input Formatting
npm install cleave.js

---

## Server Side - Express Js

# Initialize a Node.js Project
npm init -y

# Install Express
npm install express

# Install Nodemon for Development
npm install --save-dev nodemon

# Install Body Parser for Request Parsing
npm install body-parser

# Install Cors for Cross-Origin Resource Sharing
npm install cors

# Install Mongoose for MongoDB
npm install mongoose

# Install Morgan for Logging
npm install morgan

# Install bcrypt for Password Hashing
npm install bcrypt

# Install JSON Web Token (JWT) for Authentication
npm install jsonwebtoken

# Install dotenv for Environment Variables
npm install dotenv
